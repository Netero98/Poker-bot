# Аудит безопасности Poker-bot
## Высокий риск


### 5. Уязвимая версия PyJWT
**Файлы:**
* `requirements_mac.txt:26`
* `requirements_win.txt:26`

Используется:

```txt
PyJWT==1.7.0
```
Известная уязвимость:

* **CVE-2022-29217**

**Риск:**
Потенциальный bypass проверки токенов в определённых сценариях.

---

## Средний риск

### 7. Использование eval() на данных с сервера

**Файл:** `poker/tools/game_logger.py:151`

Обнаружено:

```python
eval(i)
```

где `i` приходит с удалённого API.

**Риск:**
При компрометации сервера возможен Remote Code Execution.

**Рекомендация:**
Заменить на:

```python
ast.literal_eval()
```

---

### 8. Неавторизованный endpoint `/take_screenshot`

**Файл:** `poker/restapi_local.py:34-39`

Любой локальный процесс может вызвать endpoint и сделать скриншот экрана пользователя.

**Риск:**
Локальное шпионское ПО или вредоносные процессы могут использовать API для скрытого получения изображений экрана.

---

### 9. Некорректная CORS-конфигурация

**Файл:** `restapi_local.py:26-31`

Обнаружено:

```python
allow_origins=["*"]
allow_credentials=True
```

**Риск:**
Нарушение CORS security model, потенциальное расширение поверхности атак.

---

### 10. Docker container с host network + X11 socket

Контейнер запускается с:

* `network_mode: host`
* проброшенным X11 socket

**Риск:**
Контейнер получает:

* полный сетевой доступ хоста;
* доступ к GUI/X11;
* возможность взаимодействия с окнами и вводом пользователя.

## Низкий риск / архитектурные замечания

Низкий риск, так как это явно placeholder-значения.

### 14. Лишняя зависимость `deps`

**Файл:** `package.json`

Пакет выглядит как пустышка и рекомендуется к удалению для сокращения attack surface.

# Общий вывод

Признаков вредоносного ПО, скрытого майнера или намеренно встроенного spyware не обнаружено. Проект выглядит как легитимный покерный бот с клиент-серверной архитектурой.

Основные риски связаны не со злым умыслом разработчика, а с небезопасными архитектурными решениями:

* небезопасный локальный API;
* чрезмерно открытая Docker-конфигурация.
